## Nói sơ qua

Mình học K8s là do nhiều công ty giờ đòi K8s nhiều quá. Nên của mình trước là Cloud AWS dự án nhỏ và vừa nên không chọn k8s được. 

Cấu hình và techstack sơ bộ 
- Proxmox: 8vCPU + 24GB RAM
- 1 VM Opnsense làm router
- 1 VM bastion để SSH
- 1 VM k3sServer làm control panel node
- 2 VM K3sworker làm worker node
