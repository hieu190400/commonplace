## Nói sơ qua

Mình học K8s là do nhiều công ty giờ đòi K8s nhiều quá. Mình trước là chủ yếu là Cloud AWS dự án nhỏ và vừa nên không chọn k8s được. 

Cấu hình và techstack sơ bộ 
- Proxmox: 8vCPU + 24GB RAM
- 1 VM Opnsense làm router: 2vCPU + 2GB RAM
- 1 VM bastion để SSH: 2vCPU + 0.5GB RAM
- 1 VM k3sServer làm control panel node: 2vCPU + 4GB RAM
- 2 VM K3sworker làm worker node: mỗi thằng 2vCPU + 4GB RAM

Repo: https://github.com/hieu190400/gitops-homelab
